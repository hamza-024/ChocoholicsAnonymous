Project 2 Analysis Description:

Now that we are familiar with ChocAn we can work to computerize the system and make sure it works with other systems that need to be accessed. The main systems that will be computerized are the provider, operator, and manager sections by creating a program that opens a specific terminal. Each terminal will have user-specific functions which will be determined by having the user login and verify their role. All data in each terminal will be backed up to a database. Each user will be given a laptop to run the programs. Furthermore, multiple APIs will be used to help the systems interact and share data.
